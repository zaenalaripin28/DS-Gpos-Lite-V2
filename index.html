Implementation Guide

Tiga langkah untuk mulai menggunakan design system ini di project kamu.

1 Copy Tailwind Config

Copy file tailwind.config.js ke projectmu. Semua design tokens sudah terdefinisi sebagai Tailwind theme extensions.

// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        primary: { 500: '#1E7FD6', 600: '#1668B0', ... },
        secondary: { 500: '#06B6D4', ... },
        // success, warning, error, info, gray
      },
      spacing: { 1: '0.25rem', 4: '1rem', 8: '2rem', ... },
      borderRadius: { md: '0.5rem', lg: '0.75rem', ... },
    }
  }
}

2 Gunakan Tailwind Classes

Semua tokens otomatis jadi Tailwind utilities. Copy class langsung dari dokumentasi:

<!-- Primary Button -->
<button class="px-5 py-2.5 bg-primary-500 text-white font-semibold
               rounded-md shadow-brand hover:bg-primary-600 transition-all">
  Primary Button
</button>

<!-- Card Component -->
<div class="bg-white border border-gray-200 rounded-lg p-6 shadow-sm
            hover:shadow-md hover:-translate-y-0.5 transition-all">
  Card content
</div>

3 Gunakan CSS Variables

Atau akses token langsung via CSS custom properties untuk custom styling:

.my-component {
  background-color: var(--color-primary-500);
  padding: var(--spacing-4) var(--spacing-6);
  border-radius: var(--border-radius-md);
  box-shadow: var(--shadow-brand);
  font-family: var(--font-primary);
  transition: all var(--transition-base);
}

Best Practices

Panduan singkat untuk menjaga konsistensi dan kualitas implementasi di seluruh project.

Do — Yang harus dilakukan
✅ Gunakan design tokens dari system — jangan hardcode nilai warna atau spacing
✅ Konsistenkan spacing menggunakan spacing scale 4px base unit
✅ Gunakan semantic color naming: primary, success, warning, error
✅ Ikuti typography scale dan gunakan font Inter untuk konsistensi
✅ Test tampilan pada semua breakpoints: mobile, tablet, dan desktop
Don't — Yang harus dihindari
❌ Jangan hardcode seperti #1e7fd6 atau 16px langsung di CSS
❌ Jangan membuat custom color palette di luar design system
❌ Jangan skip accessibility — pastikan contrast ratio memenuhi WCAG AA
