# GPOS Lite Visual Style Guide

Reference:

* index.html = primary visual source

## Visual Language

* Premium SaaS dashboard aesthetic
* Clean hierarchy
* Soft surface layering
* Spacious layout rhythm
* Controlled density
* Consistent card styling
* Consistent border radius
* Consistent spacing rhythm
* Subtle hover interaction
* Subtle focus interaction

## Layout Style

* Large section spacing
* Clean content grouping
* Stable visual rhythm
* Balanced whitespace
* Avoid crowded layout

## Card Style

* Soft border
* Minimal shadow
* Consistent padding
* Consistent radius
* Clean internal spacing

## Typography

* Clear hierarchy
* Comfortable readability
* Consistent heading rhythm
* Consistent label sizing

## Interaction

* Subtle hover state
* Smooth transition
* Minimal motion
* Clear active/focus state

## Avoid

* Visual overdesign
* Excessive gradients
* Heavy shadows
* Inconsistent spacing
* Random radius
* Different visual language
* Crowded sections
* Aggressive animation

## Priority

1. Existing token system
2. Existing index.html style
3. Existing component structure
4. Existing responsive behavior sampai ke level mobile
