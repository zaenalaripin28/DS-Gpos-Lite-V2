tab-part-default

Part Default — State

Item tab tidak terpilih. Referensi Figma: figma/tab:part:default.png. Gunakan .ds-tab dengan modifier --hover, --press, atau --focus untuk dokumentasi state statis.

Default
.ds-tab
Label Color-Text-Subtle. Tanpa indikator bawah.
Hover
.ds-tab--hover
Indikator abu muncul di bawah label.
Press
.ds-tab--press
Label Color-Text-Bold + indikator abu.
Focus
.ds-tab--focus
Ring fokus Border.Focused (B200).

Demo Interaktif

Arahkan kursor, klik, dan tekan Tab pada tombol di bawah.

Referensi Figma

Kode

Copy HTML untuk tab-part-default. Letakkan beberapa item dalam role="tablist".

tabs.html 
<div role="tablist" aria-label="Sections">
  <button type="button" class="ds-tab" role="tab" aria-selected="false">
    <span class="ds-tab__label">Tab</span>
    <span class="ds-tab__indicator" aria-hidden="true"></span>
  </button>
</div>
