Time Picker

Time picker component terdiri dari select trigger + panel time parts. Implementasi menggunakan pattern Select existing, time parts existing, icon asset existing, dan token dari token.css.

Time Picker Component

Informasi Penggunaan

Kapan Digunakan
  • Gunakan saat user harus memilih jam spesifik dari daftar waktu terstruktur.
  • Gunakan interval konsisten (contoh 30 menit) untuk mempercepat pemilihan.
  • Gunakan varian subtle untuk konteks form yang visualnya lebih ringan.
Catatan Interaksi
  • State focus wajib terlihat jelas di trigger untuk akses keyboard.
  • Item terpilih harus tetap terlihat saat list di-scroll.
  • Pastikan area klik item minimal setinggi trigger agar nyaman dipilih.

Do & Don’t

Do — Lakukan ini
  • Gunakan format waktu konsisten (contoh: 9:00 AM, 9:30 AM).
  • Tampilkan state terpilih menggunakan token selected existing.
  • Gunakan icon dropdown dan clear dari assets existing.
Don’t — Hindari ini
  • Jangan campur format 12-jam dan 24-jam dalam satu komponen.
  • Jangan gunakan warna manual di luar semantic token.
  • Jangan hilangkan state focus atau hover pada item waktu.
