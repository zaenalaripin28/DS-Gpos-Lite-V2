Date Picker

Komposisi Select trigger + Calendar berdasarkan referensi Date picker/figma/date picker-components.png.

December 2022

SUNMONTUEWEDTHUFRISAT

Informasi Penggunaan

  • Gunakan Date Picker untuk input tanggal tunggal yang membutuhkan akurasi pilihan hari, bulan, dan tahun.
  • Tampilkan nilai tanggal terpilih di trigger agar status input tetap terlihat tanpa membuka panel kalender.
  • Pertahankan format tanggal yang konsisten pada satu konteks (contoh: 12 December 2022).
  • Pakai saat field tidak cocok untuk input bebas agar mengurangi risiko format tanggal yang salah.

Do & Don't

Do — Lakukan ini
  • Gunakan trigger dengan ikon kalender untuk memperjelas tipe input.
  • Tandai tanggal terpilih dengan state selected yang konsisten.
  • Gunakan token semantik untuk warna, border, spacing, dan typography.
  • Pastikan fokus keyboard terlihat jelas pada trigger dan sel tanggal.
Don't — Hindari ini
  • Jangan hardcode warna atau spacing di luar token existing.
  • Jangan sembunyikan state selected sehingga pengguna bingung tanggal aktif.
  • Jangan gunakan format tanggal berbeda-beda dalam satu halaman.
  • Jangan menonaktifkan focus ring karena menurunkan aksesibilitas keyboard.
