Toggle — Components

Switch on/off dengan ikon status (icon-check.svg · icon-x-mark.svg). Dua ukuran: size=regular (default) dan size=large. Empat state interaksi. Referensi: Toggle/figma/Toggle-component.png.

Interaktif (native switch)

Anatomy

Track (pill) + thumb (knob putih) + status icons (check kiri saat on, x kanan saat off). Tanpa label teks — pasangkan label terpisah jika diperlukan.

Informasi Penggunaan

Kapan dipakai
Untuk pengaturan on/off tunggal yang berlaku segera (notifikasi, fitur aktif/nonaktif, mode tampilan). Satu toggle = satu boolean; tidak untuk memilih banyak opsi (gunakan checkbox/radio).
Ukuran & state
regular default untuk form padat; large untuk sentuhan lebih besar di mobile/tablet. State: default, hover (track lebih terang), focus (ring b200), disabled (track n30, ikon redup).
Aksesibilitas
Gunakan <input type="checkbox" role="switch"> dengan aria-checked. Sertakan label terlihat atau aria-label. Jangan mengandalkan warna hijau/abu saja — ikon check/x memberi petunjuk status.

Do & Don't

Do — Lakukan ini
  • Gunakan untuk satu pengaturan boolean (aktif/nonaktif) yang langsung diterapkan.
  • Pakai token: off n500, on g300, thumb n0, focus ring b200.
  • Tampilkan ikon icon-check dan icon-x-mark dari assets/icons.
  • Beri role="switch" dan perbarui aria-checked saat state berubah.
  • Pilih size=large bila target sentuh minimal ~44px diperlukan.
Don't — Hindari ini
  • Jangan gunakan toggle untuk memilih satu dari banyak opsi — pakai radio.
  • Jangan hardcode warna hex — gunakan CSS variable / alias Tailwind DS.
  • Jangan membuat switch hanya dari div tanpa input native yang dapat difokus keyboard.
  • Jangan menghilangkan ikon status jika warna track menjadi satu-satunya indikator.
  • Jangan menempatkan dua toggle berdekatan tanpa label yang membedakan fungsinya.

Code — Salin Code

toggle.html 
<!-- Snapshot: off default -->
<span class="ds-toggle ds-toggle--snapshot" aria-hidden="true">
  <span class="ds-toggle__track">
    <span class="ds-toggle__icon ds-toggle__icon--on"><img src="../../assets/icons/icon-check.svg" alt="" aria-hidden="true"></span>
    <span class="ds-toggle__thumb"></span>
    <span class="ds-toggle__icon ds-toggle__icon--off"><img src="../../assets/icons/icon-x-mark.svg" alt="" aria-hidden="true"></span>
  </span>
</span>

<!-- Interaktif -->
<label class="ds-toggle">
  <input type="checkbox" class="ds-toggle__input" role="switch" aria-checked="false">
  <span class="ds-toggle__track">...</span>
</label>

<!-- Large + checked -->
<label class="ds-toggle ds-toggle--large">
  <input type="checkbox" class="ds-toggle__input" role="switch" checked>
  ...
</label>
