Anatomy

Calendar dirakit dari tiga part: month header (navigasi bulan), week header (label hari), dan day cell (sel tanggal).

December 2022

A Month header
B Label
Month header
.ds-calendar-month-header — nav kiri/kanan + label Bulan YYYY. Ikon chevron dari assets/icons.
C Day cell
Day
.ds-calendar-day 32×32px. State: default, today, hover, selected, range, outside, disabled. Grid 7 kolom sejajar dengan week header.

Informasi Penggunaan

Calendar component
Rakit month header → week header → day grid dalam .ds-calendar. Varian footer opsional untuk aksi konfirmasi (Simpan).
Month header
Navigasi bulan/tahun. Label Text.Bold, ikon Icon.Bold. Padding space-100 / space-150.
Week header
Baris 7 kolom label hari (SUN–SAT). Typography overline-medium, warna Text.Subtle (n300). Sejajar dengan grid day.
Day
Sel 32×32, body-small-medium. State: default, today (underline), hover/pressed, focus, selected, range, outside, disabled.
Grid alignment
Week header dan day grid memakai grid-template-columns: repeat(7, 1fr) + gap space-050 agar kolom sejajar.
Hari di luar bulan
Tanggal trailing/leading pakai --outside + disabled — Text.Subtle, non-interaktif.
Aksesibilitas
role="columnheader" pada week day, role="grid" pada days, aria-current="date" / aria-selected pada sel aktif.

Do & Don't

Do — Lakukan ini
  • Rakit parts berurutan di dalam .ds-calendar — jangan ubah urutan month → week → days.
  • Month header: format Bulan YYYY, chevron dari assets/icons.
  • Week header: singkatan 3 huruf UPPERCASE, 7 kolom sama lebar dengan day grid.
  • Day: satu state aktif per grid; today + selected boleh digabung dengan modifier.
  • Token semantik: Text.Bold, Text.Subtle, Text.Brand, Background.Selected, Border.Focused.
Don't — Hindari ini
  • Jangan letakkan footer action di tengah — rata kanan di .ds-calendar__footer.
  • Jangan ubah jumlah kolom week/day (harus 7).
  • Jangan hardcode hex — pakai token --color-neutral-* / --color-blue-*.
  • Jangan pakai nama hari penuh di week header (ruang terbatas).
  • Jangan hilangkan focus-visible pada day button dan nav month.
