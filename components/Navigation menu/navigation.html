Informasi Penggunaan

Panduan penggunaan setiap part navigation menu dalam konteks sidebar GPOS Lite.

Sidebar Navigation Expand
Header + Menu Search + Heading --no-icons (Terakhir Dibuka) + Main Menu recent + Divider + daftar menu. Parent expandable: chevron down/up + Submenu Content.
Sidebar Navigation Hide
Modifier ds-sidebar-nav-expand--hide pada komponen expand — sembunyikan teks menu, tampilkan icon saja. Toggle chevron left/right pada header.
Navigation Menu (komposisi)
isMenu=single: Main Menu saja. hide: parent + chevron-down. expand: parent -on + chevron-up + daftar Submenu Content.
Menu Search custom
Field 32px + margin vertikal 8px. Caption regular. Icon search / clear dari assets/icons/.
Divider
Pisahkan heading menu dari daftar item. Gunakan asset Divider.svg dari assets/images/.
Dot
Indikator 6×6px via dot.svg. Satu varian muted untuk semua item — tanpa state warna primary.

Do & Don't

Panduan implementasi navigation menu yang konsisten dengan anatomy parts dan token DS.

Do — Lakukan ini
  • Letakkan Menu Search custom di atas daftar menu — tinggi space-400, jarak atas/bawah space-100.
  • Susun urutan: Menu Search → Heading Menu → Divider → Main Menu → Submenu Content → Divider penutup.
  • Satu item state=selected per section pada Main Menu atau Submenu Content.
  • Submenu Content: gunakan part Dot + typography body3 regular + chevron dari Icons.
  • Navigation Menu: rakit Main Menu + Submenu Content — isMenu=single tanpa submenu, hide/expand dengan chevron down/up.
  • Parent expandable: icon Sidebar Offline -off (collapsed) / -on (expanded). Leaf single tanpa chevron trail.
  • Tambahkan icon-chevron-right.svg sebagai trail icon pada setiap Main Menu item.
  • Terapkan hover / press / selected hanya pada link Main Menu, bukan pada Divider atau Dot.
  • Set aria-current="page" pada item Main Menu yang sedang aktif.
Don't — Hindari ini
  • Jangan campur chevron-right pada parent yang expandable — gunakan chevron-down/up.
  • Jangan render Submenu Content saat isMenu=hide.
  • Gunakan icon-magnifying-glass.svg untuk Menu Search default–typing; icon-x-circle-outline.svg pada state rest.
  • Jangan ubah tinggi field dari 32px atau typography selain caption regular.
  • Jangan tambahkan varian warna primary pada komponen Dot.
  • Jangan ganti Divider dengan border pada item — divider adalah part terpisah.
  • Jangan ganti dot.svg dengan span CSS atau ukuran selain 6×6px.
  • Jangan terapkan state selected pada Heading Menu atau Divider.
  • Jangan hardcode warna hex — gunakan token border-subtle, accent-primary, surface-hover.

Playground

Toggle sidebar expand / hide lewat tombol chevron. Saat hide, label menu disembunyikan (icon saja). Klik item untuk state=selected; parent expandable untuk submenu saat expand.
