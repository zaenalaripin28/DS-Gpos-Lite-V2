Anatomy

Pagination terdiri dari wrapper nav, deretan item (tombol halaman / navigasi), dan opsional ikon chevron pada previous/next.

Informasi Penggunaan

Kapan dipakai
Gunakan untuk navigasi antar halaman list/table ketika jumlah data besar dan tidak muat dalam satu layar.
State aktif
Selalu tandai halaman aktif dengan background Color-Background-Brand-Subtle dan teks brand.
Aksesibilitas
Gunakan nav[aria-label], aria-current="page" pada item aktif, dan label untuk tombol previous/next.

Do & Don't

Do — Lakukan ini
  • Tampilkan satu halaman aktif pada satu waktu.
  • Pakai ellipsis saat jumlah halaman panjang.
  • Gunakan ikon chevron existing di assets/icons.
  • Jaga jarak antar item konsisten dengan spacing token.
Don't — Hindari ini
  • Jangan beri dua item aktif bersamaan.
  • Jangan hardcode warna hex di komponen.
  • Jangan ganti ikon dengan library eksternal.
  • Jangan hilangkan focus state untuk keyboard user.
