Informasi Penggunaan

Kapan dipakai
Edit cepat nilai inline pada tabel, metadata, atau inline message — tanpa membuka modal/form penuh.
Trigger Select
Menu inline edit hanya tampil pada state-focus, state-typing, dan state-empty. State lain (default, hover, disabled) menyembunyikan menu.
Select inline message
Gunakan .ds-select-trigger--inline-message — tanpa ikon chevron di UI. Tombol confirm/cancel memakai .ds-inline-edit-btn.

Do & Don't

Do — Lakukan ini
  • Tampilkan menu inline edit hanya saat select pada focus, typing, atau empty.
  • Gunakan select inline message tanpa ikon — fokus pada teks dan aksi edit.
  • Sertakan aria-label pada tombol confirm dan cancel.
  • Gunakan token semantic untuk border fokus, latar, dan shadow tombol.
Don't — Hindari ini
  • Jangan tampilkan menu edit saat select dalam state default/hover/disabled.
  • Jangan tambahkan ikon chevron pada select inline message.
  • Jangan hardcode warna hex — gunakan tokens.css.
  • Jangan biarkan menu edit tetap terbuka setelah confirm/cancel tanpa logika aplikasi.
