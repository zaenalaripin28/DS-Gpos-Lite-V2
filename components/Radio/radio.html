Radio — Group

Kumpulan opsi vertikal — tiap baris memakai komponen .ds-radio dengan name yang sama sehingga hanya satu terpilih. Jarak antar item space-150 (12px). Referensi: Radio/figma/radiobutton-group-component.png.

Interaktif (fieldset + legend)

Pilih opsi
Struktur grup
Bungkus dengan <fieldset class="ds-radio-group">. Semua input[type=radio] dalam grup memakai name identik. Opsi pertama boleh checked sebagai default.
Layout
Susun vertikal (flex-direction: column). Minimum 2 opsi. Untuk banyak pilihan pertimbangkan select atau segmented control.

Anatomy

Radio item = lingkaran 16×16 + label. Radio group = container vertikal berisi beberapa item dengan jarak tetap.

Informasi Penggunaan

Kapan dipakai
Untuk memilih tepat satu opsi dalam grup yang saling eksklusif (metode pembayaran, jenis akun, pengaturan default). Gunakan checkbox jika pengguna boleh memilih lebih dari satu.
Parts & komponen
Part <RadioIcon>; komponen tunggal .ds-radio; grup .ds-radio-group = beberapa .ds-radio vertikal dengan name sama.
Aksesibilitas
Gunakan <input type="radio"> dengan name yang sama per grup. Bungkus dengan <fieldset> + <legend> untuk konteks grup. Sertakan pesan teks saat aria-invalid.

Do & Don't

Do — Lakukan ini
  • Gunakan lingkaran 16×16px (rounded-full) dan titik dalam 6px untuk state checked.
  • Kelompokkan radio dalam .ds-radio-group dengan name identik agar hanya satu terpilih.
  • Susun grup secara vertikal; gunakan legend jika grup butuh judul yang jelas.
  • Hover unchecked: fill n20; press: n40; focus: ring b200 2px.
  • Checked memakai Background.Brand.Bold (b300) + titik Icon.Inverse (n0).
  • Invalid pada checked: ring luar Border.Danger (r300), bukan mengganti fill brand.
Don't — Hindari ini
  • Jangan membuat radio hanya dari div tanpa input type="radio" native.
  • Jangan hardcode warna hex — gunakan token CSS atau alias Tailwind DS.
  • Jangan membiarkan beberapa radio dalam grup terpilih bersamaan (salah name).
  • Jangan gunakan radio untuk aksi on/off tunggal — gunakan toggle/checkbox.
  • Jangan menampilkan ring error sebelum pengguna berinteraksi atau submit form.

Code — Salin Code

radio.html 
<!-- Part: checked (static) -->
<span class="ds-radio__circle ds-radio__circle--checked" aria-hidden="true">
  <span class="ds-radio__dot" style="display:block;"></span>
</span>

<!-- Komponen: label + input -->
<label class="ds-radio">
  <input type="radio" class="ds-radio__input" name="opsi" value="a">
  <span class="ds-radio__circle" aria-hidden="true"><span class="ds-radio__dot"></span></span>
  <span class="ds-radio__label">Label</span>
</label>

<!-- Radio group -->
<fieldset class="ds-radio-group">
  <legend class="ds-radio-group__legend">Pilih opsi</legend>
  <label class="ds-radio">
    <input type="radio" class="ds-radio__input" name="opsi-grup" value="1" checked>
    <span class="ds-radio__circle" aria-hidden="true"><span class="ds-radio__dot"></span></span>
    <span class="ds-radio__label">Label</span>
  </label>
  <label class="ds-radio">...</label>
</fieldset>

<!-- Checked + invalid -->
<label class="ds-radio ds-radio--error">
  <input type="radio" class="ds-radio__input" name="opsi" checked aria-invalid="true">
  ...
</label>
