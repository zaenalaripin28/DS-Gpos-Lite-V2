Do & Don't

Panduan penggunaan breadcrumb yang benar untuk menjaga konsistensi navigasi dan keterbacaan hierarki.

Do — Lakukan ini
Home / Products / Detail
Gunakan untuk hierarki ≥ 2 level
Breadcrumb mulai bermakna saat ada minimal 2 level navigasi. Satu item saja tidak perlu breadcrumb.
Halaman Saat Ini
Tampilkan current page di state Focus
Item terakhir (halaman aktif) gunakan state focus — warna subtle N300, tidak perlu link.
Label Pendek
Gunakan label ringkas per item
Label breadcrumb idealnya 1–3 kata. Jika panjang, aktifkan truncationWidth=true.
Home / Current
Bungkus dengan <nav aria-label>
Selalu gunakan elemen <nav> dengan aria-label="Breadcrumb" untuk aksesibilitas screen reader.
Don't — Hindari ini
Beranda / Produk / Kategori / Sub / Detail / Item A
Jangan tampilkan terlalu banyak level
Lebih dari 4–5 level membingungkan pengguna. Pertimbangkan menyederhanakan atau menyembunyikan level tengah.
Halaman Ini
Jangan jadikan item current sebagai link aktif
Item terakhir adalah halaman yang sedang dibuka — tidak boleh diklik. Gunakan state focus, bukan default.
Home Produk Detail
Jangan ganti separator dengan ikon chevron
Separator komponen ini adalah karakter /. Jangan diganti chevron ikon — itu bagian dari slot trailing icon, bukan separator.
Breadcrumb
Jangan ubah warna di luar token yang ditentukan
Semua warna harus dari semantic token: B300 (link), B400 (pressed), N300 (subtle). Jangan hardcode warna kustom.

Kode

Copy HTML langsung dari sini. Gunakan class .ds-bc-item + modifier state sebagai base.

breadcrumbs.html 
<!-- state=default, truncationWidth=false -->
<div class="ds-bc-item ds-bc-item--default">
  <span class="ds-bc-sep">/</span>
  <div class="ds-bc-container">
    <div class="ds-bc-icon"><img src="icon-home.svg" alt=""></div>
    <span class="ds-bc-text">Item</span>
    <div class="ds-bc-icon"><img src="icon-chevron-right.svg" alt=""></div>
  </div>
</div>

<!-- state=default, truncationWidth=true -->
<div class="ds-bc-item ds-bc-item--default">
  <span class="ds-bc-sep">/</span>
  <div class="ds-bc-container ds-bc-container--truncated">
    <div class="ds-bc-icon"><img src="icon-home.svg" alt=""></div>
    <span class="ds-bc-text ds-bc-text--truncated">Item Sale</span>
    <div class="ds-bc-icon"><img src="icon-chevron-right.svg" alt=""></div>
  </div>
</div>

<!-- state=hover | gunakan --hover, --press, atau --focus sebagai modifier -->
<div class="ds-bc-item ds-bc-item--hover"> ... </div>
<div class="ds-bc-item ds-bc-item--press"> ... </div>
<div class="ds-bc-item ds-bc-item--focus"> ... </div>
