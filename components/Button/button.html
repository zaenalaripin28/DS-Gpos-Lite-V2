Komponen Siap Produksi

Copy-ready button menggunakan .ds-btn beserta semua state interaktif. Tambahkan modifier class untuk mengubah tampilan.

CSS — Kelas dasar & modifier tampilan (tokens.css)
/* Kelas dasar */
.ds-btn {
  background-color: var(--color-neutral-n20);
  color:            var(--color-neutral-n900);
  font-family:      var(--font-primary);       /* Poppins */
  font-size:        var(--font-size-sm);       /* 14px */
  font-weight:      var(--font-weight-medium); /* 500 */
  padding:          0 var(--space-150);        /* 0 12px */
  min-height:       32px;
  border-radius:    6px;
}
/* Modifier tampilan */
.ds-btn--primary    { background-color: var(--color-blue-b300);    color: var(--color-neutral-n0); }
.ds-btn--subtle     { background-color: transparent; }
.ds-btn--link       { background-color: transparent; color: var(--color-blue-b300); }
.ds-btn--danger     { background-color: var(--color-red-r300);     color: var(--color-neutral-n0); }
.ds-btn--warning    { background-color: var(--color-orange-o200);  color: var(--color-neutral-n900); }
/* Modifier ukuran & bentuk */
.ds-btn--compact    { min-height: 24px; }
.ds-btn--icon-only  { padding: 0 var(--space-050); gap: 0; }
.ds-btn--loading    { pointer-events: none; cursor: default; }
HTML — Pola penggunaan umum
<!-- Default (aksi sekunder) -->
<button type="button" class="ds-btn">Batal</button>

<!-- Primary (aksi utama) -->
<button type="button" class="ds-btn ds-btn--primary">Simpan</button>

<!-- Kompak -->
<button type="button" class="ds-btn ds-btn--compact">Filter</button>

<!-- Dipilih / toggle -->
<button type="button" class="ds-btn" aria-pressed="true">Dipilih</button>

<!-- Nonaktif -->
<button type="button" class="ds-btn ds-btn--primary" disabled>Nonaktif</button>

<!-- Hanya Ikon -->
<button type="button" class="ds-btn ds-btn--icon-only" aria-label="Hapus"><!-- svg --></button>

<!-- Loading -->
<button type="button" class="ds-btn ds-btn--primary ds-btn--loading" aria-busy="true">
  <span class="ds-spinner"><!-- svg spinner --></span>
</button>

Panduan Penggunaan

Aturan praktis untuk memilih tampilan dan status yang tepat agar tombol dapat diakses, konsisten, dan mudah digunakan.

Lakukan
Gunakan atribut disabled pada elemen <button> agar pembaca layar mengumumkan status nonaktif secara native tanpa JavaScript tambahan.
Jangan
Jangan gunakan <div> atau <a> untuk pemicu aksi — selalu gunakan elemen <button> yang semantik agar keyboard dan pembaca layar berfungsi dengan benar.
Lakukan
Tambahkan aria-pressed="true" saat isSelected=true untuk mengomunikasikan status toggle kepada teknologi bantu secara akurat.
Jangan
Jangan hilangkan fokus dengan outline:none tanpa menyediakan indikator fokus pengganti — .ds-btn:focus-visible sudah menangani ini dengan benar.
Lakukan
Gunakan hanya satu tombol appearance=primary per halaman atau per area aksi — primary menandai satu aksi paling penting dalam konteksnya.
Jangan
Jangan tempatkan dua tombol primary berdampingan — dua CTA merah/biru yang bersaing membingungkan pengguna tentang langkah yang harus diambil selanjutnya.
Lakukan
Selalu sertakan aria-label pada tombol ikon-saja (.ds-btn--icon-only) — pengguna pembaca layar memerlukan teks deskriptif untuk memahami fungsi tombol.
Jangan
Jangan gunakan appearance=danger untuk aksi yang tidak destruktif — warna merah harus dicadangkan untuk aksi ireversibel seperti hapus atau reset data.
