Button

Triggers an action or event. Follows the Figma variant architecture with appearance, state, spacing, and boolean properties.

Interactive States

Live interactive button — hover, click, and tab to observe each state.

Usage Guidance

Do
Use disabled attribute on the <button> element so screen readers announce the disabled state natively.
Don't
Never use <div> or <a> for action triggers — use a semantic <button>.
Do
Add aria-pressed="true" when isSelected=true to communicate toggle state to assistive technology.
Don't
Don't suppress focus with outline:none without providing a custom focus indicator.
