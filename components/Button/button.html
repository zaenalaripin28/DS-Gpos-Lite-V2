Usage Guidelines
✅Gunakan Primary untuk satu aksi utama per halaman: Simpan, Submit, Konfirmasi.
❌Jangan gunakan lebih dari satu Primary button dalam satu area yang sama.
✅Gunakan Ghost untuk aksi sekunder/tersier. Background neutral ringan — tidak mendominasi.
❌Jangan gunakan White atau Secondary di atas background terang — dirancang khusus untuk dark background.
✅Gunakan label yang jelas & actionable: "Simpan Data", "Hapus", "Kirim" — bukan hanya "OK".
❌Jangan disable button tanpa memberikan tooltip atau pesan kenapa ia tidak bisa diklik.

Elemen Anatomy
#ElemenKeterangan
1 Container Frame dengan border-radius 6px, background neutral-n20 (#F5F6F7). Ukuran otomatis menyesuaikan konten (hug). Min-height ~36px.
2 Leading Icon Opsional. Ukuran 14×14px. Gap ke label: 4px. Stroke 2–2.5px. Gunakan SVG atau icon system.
3 Label Wajib ada. Font Poppins Medium 14px, line-height 20px. Color neutral-n900 (#091E42). Gunakan kalimat actionable.
4 Trailing Icon Opsional. Sama seperti Leading Icon. Gunakan untuk direktional (arrow, chevron) atau status.
Accessibility
♿ Keyboard Navigation — :focus-visible

Gunakan :focus-visible agar focus ring hanya tampil saat navigasi keyboard (Tab), tidak saat klik mouse. Focus ring Ghost: box-shadow: 0 0 0 3px rgba(30,127,214,0.25)

✅ Contrast Ratio — WCAG AA

neutral-n900 (#091E42) pada neutral-n20 (#F5F6F7) — contrast ratio 11.8:1, jauh di atas WCAG AA (min 4.5:1). Primary white text pada #1E7FD6: 4.6:1 — memenuhi WCAG AA.

⚠️ Disabled State

Gunakan aria-disabled="true" + tabindex="-1" jika ingin button tetap focusable dengan screen reader. Jangan hanya mengandalkan opacity visual.

/* Ghost Neutral — Figma: Color-Background-Neutral · Color-Text-Bold */
.btn-ghost-fixed {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  gap: 4px;                          /* Figma: gap 4px */
  padding: 8px 12px;                 /* Figma: px 12px, py auto */
  border-radius: 6px;                /* Figma: border-radius 6px */
  font-family: var(--font-primary);
  font-size: var(--font-size-sm);    /* 14px */
  font-weight: var(--font-weight-medium); /* 500 */
  line-height: 20px;                 /* Figma: line-height 20px */
  border: none;
  background-color: var(--color-neutral-n20);  /* #F5F6F7 */
  color: var(--color-neutral-n900);            /* #091E42 */
  transition: all var(--transition-base);
  cursor: pointer;
  white-space: nowrap;
  text-decoration: none;
}

.btn-ghost-fixed:hover {
  background-color: var(--color-neutral-n30);  /* #EBEDF0 */
}
.btn-ghost-fixed:active {
  background-color: var(--color-neutral-n40);  /* #DFE2E6 */
  transform: translateY(1px) scale(.98);
}
.btn-ghost-fixed:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px rgba(30, 127, 214, .25);
}
.btn-ghost-fixed:disabled {
  opacity: .4;
  cursor: not-allowed;
  pointer-events: none;
}
