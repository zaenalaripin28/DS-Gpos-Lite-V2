Grid & Layout System

4 breakpoint responsif — Small (4 col) · Medium (8 col) · Large (12 col) · XLarge (12 col fixed) — dengan margin 24px, gutter 16px, untuk layout konsisten di semua ukuran layar.

Ringkasan Breakpoint

Tier Breakpoint Kolom Margin Gutter Tipe
Small 320 – 600px 4 24px 16px Fluid / Stretched
Medium 601 – 1024px 8 24px 16px Fluid / Stretched
Large 1025 – 1440px 12 24px 16px Fluid / Stretched
XLarge 1400px+ 12 Auto 16px Fixed / Centre

Small Grid

4 columns SM

Grid untuk mobile devices (320–600px). Terdiri dari 4 kolom fluid dengan margin 24px dan gutter 16px. Digunakan untuk layout utama smartphone portrait.

📌 Kapan Dipakai
  • Layout utama smartphone portrait
  • Card list vertikal satu kolom
  • Form input full-width
  • Bottom navigation bar content
  • Modals & drawer mobile
✅ Best Practice
  • Mulai selalu dari 4 kolom (mobile-first)
  • Gunakan padding 24px kiri-kanan konsisten
  • 2-column span untuk card medium (col-span-2)
  • Touch target minimal 44×44px
✖ Hindari
  • Jangan skip ke 8/12 kolom langsung di mobile
  • Jangan kurangi margin di bawah 16px
  • Hindari teks terlalu panjang satu baris
  • Jangan pakai fixed width pada elemen

12 columns LG

✅ Best Practice

  • Hindari lebih dari 12 kolom dalam satu baris
  • Jangan skip breakpoint MD ke LG langsung
  • Hindari fixed width pada kolom fluid

Extra Large Grid

12 columns · fixed XL

Grid fixed-centre untuk layar ultrawide (1400px+). Konten dikunci di tengah dengan lebar kolom 100px dan margin auto, agar mudah dibaca tanpa konten meregang terlalu jauh.

  • Konten fixed-width agar mudah dibaca
  • Landing page premium & marketing
  • Dashboard data-heavy dengan banyak kolom
  • Layout yang mengutamakan white space
✅ Best Practice
  • 12 kolom × 100px + 11 gutter = max ±1376px konten
  • Selalu centering dengan margin-inline: auto
  • Manfaatkan white space di kiri & kanan
  • Pertahankan konsistensi 16px gutter
✖ Hindari
  • Jangan stretch konten full-width di atas 1400px
  • Hindari fluid grid di breakpoint XL
  • Jangan abaikan white space samping
  • Hindari fixed margin angka di XL (gunakan auto)
Implementation Code
/* ── Mobile-First Responsive Grid ─────────────────────── */ /* Small (default) — 4 kolom fluid */ <div class="grid grid-cols-4 gap-4 px-6"> <div class="col-span-4">Full width</div> <div class="col-span-2">Half</div> <div class="col-span-2">Half</div> </div> /* Medium (sm:) — 8 kolom fluid, tablet */ <div class="grid grid-cols-4 sm:grid-cols-8 gap-4 px-6"> <div class="col-span-4 sm:col-span-4">Main</div> <div class="col-span-4 sm:col-span-4">Sidebar</div> </div> /* Large (lg:) — 12 kolom fluid, desktop */ <div class="grid grid-cols-4 sm:grid-cols-8 lg:grid-cols-12 gap-4 px-6"> <div class="col-span-4 sm:col-span-8 lg:col-span-8">Content</div> <div class="col-span-4 sm:col-span-8 lg:col-span-4">Sidebar</div> </div> /* XLarge (xl:) — 12 kolom fixed, centred */ <div class="max-w-screen-xl mx-auto grid grid-cols-12 gap-4"> <div class="col-span-8">Main Content</div> <div class="col-span-4">Sidebar</div> </div> /* ── Common Column Span Patterns ─────────────────────── */ /* Full width */ "col-span-12" /* Half (6+6) */ "col-span-6" /* 8+4 pattern */ "col-span-8" + "col-span-4" /* 3 columns */ "col-span-4" /* 4 columns */ "col-span-3"
/* ── Grid Token Definitions ──────────────────────────── */ :root { /* Breakpoints */ --grid-bp-sm: 320px; /* Small start */ --grid-bp-md: 601px; /* Medium start */ --grid-bp-lg: 1025px; /* Large start */ --grid-bp-xl: 1400px; /* XLarge start */ /* Column counts */ --grid-columns-sm: 4; --grid-columns-md: 8; --grid-columns-lg: 12; --grid-columns-xl: 12; /* Spacing */ --grid-gutter: 16px; /* Antar kolom */ --grid-margin: 24px; /* SM / MD / LG */ --grid-margin-xl: auto; /* XL auto-centre */ --grid-col-xl-width: 100px; /* Fixed per kolom XL */ } /* ── Grid Container Utilities ─────────────────────────── */ /* Small — 4 col fluid */ .grid-sm { display: grid; grid-template-columns: repeat(var(--grid-columns-sm), 1fr); gap: var(--grid-gutter); padding-inline: var(--grid-margin); } /* Medium — 8 col fluid */ @media (min-width: 601px) { .grid-responsive { grid-template-columns: repeat(var(--grid-columns-md), 1fr); } } /* Large — 12 col fluid */ @media (min-width: 1025px) { .grid-responsive { grid-template-columns: repeat(var(--grid-columns-lg), 1fr); } } /* XLarge — 12 col fixed, centred */ @media (min-width: 1400px) { .grid-xl { display: grid; grid-template-columns: repeat(var(--grid-columns-xl), var(--grid-col-xl-width)); gap: var(--grid-gutter); margin-inline: var(--grid-margin-xl); } }
