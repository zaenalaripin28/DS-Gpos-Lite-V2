Icon System

146 icon SVG tersedia dalam 3 kategori — General, Sidebar Offline, dan Sidebar Online. Semua icon dipanggil via file path <img src>, tanpa inline SVG.

Icon Guidelines

RULES

Aturan penggunaan icon untuk menjaga konsistensi, performa, dan kemudahan maintenance di seluruh produk GPOS Lite.

📌 Kapan Dipakai
  • Gunakan icon untuk memperkuat label, bukan menggantikannya
  • Cocok di button, input, navigasi, dan status indicator
  • Panggil via <img src> dengan path yang benar
  • Gunakan ukuran sesuai konteks: small/medium/large/xlarge
  • Sidebar icon: gunakan state Off dan On sesuai kondisi aktif
✅ Best Practice
  • Semua file icon tersimpan di assets/icons/
  • Jangan duplikasi icon ke folder lain
  • Selalu sertakan alt yang deskriptif untuk aksesibilitas
  • Tambahkan width dan height untuk mencegah layout shift
  • Ikuti naming convention yang sudah ada
✖ Hindari
  • Jangan embed SVG path langsung di HTML
  • Jangan gunakan icon tanpa alt text
  • Jangan ubah ukuran di luar token yang tersedia
  • Jangan letakkan icon baru di luar folder assets/icons/
  • Jangan gunakan icon sebagai pengganti teks pada area penting

Implementation Code

CODE

Contoh implementasi icon dalam HTML standar, CSS Variable, dan Tailwind CSS.

Cara Penggunaan Icon

HTML 
<!-- Icon dasar (medium = 24px default) -->
<img src="assets/icons/icon-home.svg"
     alt="home"
     width="24" height="24">

<!-- Button dengan icon -->
<button class="btn btn-primary">
  <img src="assets/icons/icon-plus.svg"
       alt="tambah"
       width="20" height="20">
  Tambah Item
</button>

<!-- Sidebar icon — state Off (tidak aktif) -->
<img src="assets/icons/icon-sidebar-offline-news-off.svg"
     alt="berita"
     width="24" height="24">

<!-- Sidebar icon — state On (aktif) -->
<img src="assets/icons/icon-sidebar-offline-news-on.svg"
     alt="berita"
     width="24" height="24">
CSS 
/* Icon Size Tokens */
:root {
  --icon-size-small:  16px;
  --icon-size-medium: 24px;  /* default */
  --icon-size-large:  32px;
  --icon-size-xlarge: 48px;
}

/* Utility classes */
.icon    { width: var(--icon-size-medium); height: var(--icon-size-medium); }
.icon-sm { width: var(--icon-size-small);  height: var(--icon-size-small); }
.icon-lg { width: var(--icon-size-large);  height: var(--icon-size-large); }
.icon-xl { width: var(--icon-size-xlarge); height: var(--icon-size-xlarge); }
Tailwind CSS 
<!-- small 16px -->
<img src="assets/icons/icon-home.svg" alt="home" class="w-4 h-4">
<!-- medium 24px (default) -->
<img src="assets/icons/icon-home.svg" alt="home" class="w-6 h-6">
<!-- large 32px -->
<img src="assets/icons/icon-home.svg" alt="home" class="w-8 h-8">
<!-- xlarge 48px -->
<img src="assets/icons/icon-home.svg" alt="home" class="w-12 h-12">

<!-- Button dengan icon -->
<button class="flex items-center gap-2 px-4 py-2
         bg-blue-600 text-white rounded-lg font-medium">
  <img src="assets/icons/icon-plus.svg"
       alt="tambah" class="w-5 h-5">
  Tambah Produk
</button>
