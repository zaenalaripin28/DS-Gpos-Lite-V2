Usage Guidelines

Panduan penggunaan typography yang benar dalam DS GPOS Lite V2.

📌 Kapan Dipakai
  • Heading & page title utama
  • Body text & paragraf konten
  • Label form & input fields
  • Button text & CTA
  • Navigation & menu items
✅ Best Practice
  • Gunakan Body / Medium untuk konten utama
  • Gunakan Caption untuk metadata & label kecil
  • Gunakan Header / Large untuk hero section
  • Minimal 14px untuk body text readability
❌ Hindari
  • Jangan gunakan font selain Poppins di UI
  • Jangan pakai Caption (<12px) untuk body text
  • Hindari skip level heading (H1 → H4)
  • Jangan hardcode pixel size di luar scale
