Usage Guidelines

📌 Kapan Dipakai
  • Card produk & informasi
  • Button elevasi (CTA utama)
  • Panel & sidebar floating
  • Dropdown & popover menu
  • Modal & dialog overlay
  • Input field saat fokus
✅ Best Practice
  • Gunakan satu shadow token konsisten di seluruh UI
  • Shadow memperkuat hierarchy, bukan dekorasi
  • Pastikan background putih atau terang agar shadow terlihat
  • Kombinasikan dengan border-radius untuk estetika modern
✖ Hindari
  • Jangan tumpuk shadow di atas shadow
  • Jangan pakai shadow pada background gelap
  • Jangan custom shadow di luar token ini
  • Hindari di elemen yang tidak perlu elevasi

Implementation Code

Shadow 1 shadow-1
Tailwind Config — tailwind.config.js
// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      boxShadow: {
        // Shadow 1 — Multi-layer GPOS Lite
        '1': '0px 0px 1px 0px rgba(9,30,66,0.31), 0px 8px 12px 0px rgba(9,30,66,0.15)',
      },
    },
  },
};
Cara Pakai di HTML
<!-- Gunakan class shadow-1 di element -->
<div class="shadow-1 rounded-xl bg-white p-6">
  Card dengan Shadow 1
</div>

<!-- Hover state -->
<div class="shadow-1 hover:shadow-1 transition-shadow rounded-xl">
  Card dengan hover effect
</div>
Shadow 1 var(--shadow-1)
CSS Variable — styles.css / :root
:root {
  /* Shadow 1 — Multi-layer GPOS Lite */
  --shadow-1:
    0px 0px 1px 0px rgba(9, 30, 66, 0.31),  /* Layer 1 — edge */
    0px 8px 12px 0px rgba(9, 30, 66, 0.15);  /* Layer 2 — depth */
}
Cara Pakai di CSS / HTML Style
/* Di file CSS kamu */
.card {
  box-shadow: var(--shadow-1);
  border-radius: 12px;
  background: #ffffff;
}

/* Atau langsung di atribut style HTML */
<div style="box-shadow: var(--shadow-1); border-radius: 12px;">
  Card dengan Shadow 1
</div>
