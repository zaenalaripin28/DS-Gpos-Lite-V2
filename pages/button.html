Button terdiri dari container + label. Icon bersifat opsional dan bisa di kiri atau kanan label.

Usage Guidelines
✅ Gunakan Primary untuk satu aksi utama per halaman — simpan, submit, confirm.
❌ Jangan gunakan lebih dari satu Primary button dalam satu area yang sama.
✅ Gunakan Ghost untuk aksi sekunder atau aksi yang kurang penting seperti Cancel atau Back.
❌ Jangan gunakan Secondary (border putih) pada background terang — dirancang khusus untuk dark background.
✅ Gunakan label yang jelas dan actionable: "Simpan", "Kirim", "Hapus". Bukan hanya "OK" atau "Ya".
❌ Jangan nonaktifkan button tanpa memberikan feedback kenapa ia disabled.
4 Variants — dari CSS Design System
Primary
Aksi utama. Background solid brand color #1E7FD6, teks putih, shadow brand. Gunakan untuk satu aksi paling penting.
.btn-primary Dark bg: No Light bg: Yes
Ghost
Aksi sekunder atau tersier. Background transparan, border abu, teks secondary. Dipakai untuk Cancel, Back, atau aksi kurang prioritas.
.btn-ghost Dark bg: No Light bg: Yes
White Button
White
CTA utama di atas dark background — hero section, banner, card gelap. Background putih, teks primary-600. Hover: background primary-50.
.btn-white Dark bg: Yes Light bg: No
Secondary
Secondary
Aksi sekunder di atas dark background. Outline transparan, border putih redup, teks putih. Hover: background semi-transparan putih.
.btn-secondary Dark bg: Yes Light bg: No

State Reference Table
State Trigger Visual Change CSS Property
Default Idle — tidak ada interaksi Background #1E7FD6, shadow brand --color-primary-500
Hover :hover — mouse over Background lebih gelap #1668B0, naik 1px, shadow kuat translateY(-1px)
Focus :focus-visible — keyboard / tab Focus ring biru 3px transparan di luar border box-shadow ring
Active :active — saat diklik Turun 1px, scale 98%, background primary-700 translateY(1px) scale(.98)
Disabled disabled attr atau .btn-disabled Opacity 40%, cursor not-allowed, no pointer events opacity: 0.4
Loading Ditambahkan class .btn-loading Teks hilang, spinner putih muncul di tengah via ::after ::after + animation
♿ Accessibility — Keyboard Navigation

Button harus selalu memiliki focus ring yang visible saat navigasi keyboard (Tab). Gunakan :focus-visible agar focus ring hanya tampil saat keyboard, bukan saat klik mouse. Contrast ratio teks pada state default memenuhi WCAG AA (≥ 4.5:1).

Loading State
HTML + CSS 
<!-- Tambahkan class .btn-loading saat proses berlangsung -->
<button class="btn btn-primary btn-loading">Menyimpan...</button>

/* CSS untuk loading state */
.btn-loading {
  position: relative;
  color: transparent !important;
  pointer-events: none;
}
.btn-loading::after {
  content: '';
  position: absolute;
  width: 14px; height: 14px;
  border: 2px solid rgba(255,255,255,0.35);
  border-top-color: white;
  border-radius: 50%;
  animation: btn-spin .65s linear infinite;
}
@keyframes btn-spin { to { transform: rotate(360deg); } }
Focus Ring (Accessibility)
CSS 
/* Tambahkan ke styles.css — focus ring untuk keyboard nav */
.btn:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px rgba(30,127,214,0.35), var(--shadow-brand);
}
.btn:active {
  transform: translateY(1px) scale(0.98);
}
.btn:disabled,
.btn[disabled] {
  opacity: 0.4;
  cursor: not-allowed;
  pointer-events: none;
}

CSS 
/* ── Base Button ── */
.btn {
  display: inline-flex;
  align-items: center;
  justify-content: center;
  gap: var(--spacing-2);            /* 8px */
  padding: 10px var(--spacing-5);   /* 10px 20px */
  border-radius: var(--border-radius-md); /* 8px */
  font-family: var(--font-primary); /* Poppins */
  font-size: var(--font-size-sm);   /* 14px */
  font-weight: var(--font-weight-semibold); /* 600 */
  border: 1.5px solid transparent;
  transition: all var(--transition-base); /* 200ms ease-in-out */
  cursor: pointer;
  letter-spacing: 0.01em;
  white-space: nowrap;
  line-height: 1;
  text-decoration: none;
}

/* ── Primary ── */
.btn-primary {
  background-color: var(--color-primary-500); /* #1E7FD6 */
  color: white;
  border-color: var(--color-primary-500);
  box-shadow: var(--shadow-brand);
}
.btn-primary:hover {
  background-color: var(--color-primary-600); /* #1668B0 */
  border-color: var(--color-primary-600);
  box-shadow: 0 6px 20px rgba(30,127,214,0.5);
  transform: translateY(-1px);
  color: white; text-decoration: none;
}

/* ── Ghost ── */
.btn-ghost {
  background-color: transparent;
  color: var(--color-text-secondary);
  border-color: var(--color-border);
}
.btn-ghost:hover {
  background-color: var(--color-gray-100);
  color: var(--color-text-primary);
  text-decoration: none;
}

/* ── White (dark background) ── */
.btn-white {
  background-color: white;
  color: var(--color-primary-600);
  border-color: white;
  box-shadow: var(--shadow-sm);
}
.btn-white:hover {
  background-color: var(--color-primary-50);
  box-shadow: var(--shadow-md);
  transform: translateY(-1px);
  color: var(--color-primary-700);
  text-decoration: none;
}

/* ── Secondary (dark background) ── */
.btn-secondary {
  background-color: transparent;
  color: white;
  border-color: rgba(255,255,255,0.4);
}
.btn-secondary:hover {
  background-color: rgba(255,255,255,0.1);
  border-color: rgba(255,255,255,0.65);
  transform: translateY(-1px);
  color: white; text-decoration: none;
}
