Button

Komponen interaktif untuk aksi utama, sekunder, dan tersier dalam interface.

Stable 4 Variants 6 States WCAG AA

Usage Guidelines
✅Gunakan Primary untuk satu aksi utama per halaman: Simpan, Submit, Konfirmasi.
❌Jangan gunakan lebih dari satu Primary button dalam satu area yang sama.
✅Gunakan Ghost untuk aksi sekunder atau tersier. Border tidak ditampilkan — tampak seperti teks biasa.
❌Jangan gunakan White atau Secondary di atas background terang — dirancang khusus untuk dark background.
✅Gunakan label yang jelas & actionable: "Simpan Data", "Hapus", "Kirim" — bukan hanya "OK".
❌Jangan disable button tanpa memberikan tooltip atau pesan kenapa ia tidak bisa diklik.

Elemen Anatomy
#ElemenKeterangan
1 Container Frame dengan border-radius 8px. Ukuran otomatis menyesuaikan konten (hug). Min-height 36px.
2 Leading Icon Opsional. Ukuran 14×14px. Gap ke label: 8px. Stroke 2–2.5px. Gunakan SVG atau icon system.
3 Label Wajib ada. Font Poppins SemiBold 14px, line-height 1, letter-spacing 0.01em. Gunakan kalimat actionable.
4 Trailing Icon Opsional. Sama seperti Leading Icon. Gunakan untuk direktional (arrow, chevron) atau status.
Accessibility
♿ Keyboard Navigation — :focus-visible

Gunakan :focus-visible agar focus ring hanya tampil saat navigasi keyboard (Tab), tidak saat klik mouse. Focus ring: box-shadow: 0 0 0 3px rgba(30,127,214,0.35)

✅ Contrast Ratio — WCAG AA

White text pada #1E7FD6 memiliki contrast ratio 4.6:1 — memenuhi WCAG AA (min 4.5:1). Primary-600 #1668B0 pada white: 5.7:1.

⚠️ Disabled State

Gunakan aria-disabled="true" + tabindex="-1" jika ingin button tetap focusable dengan screen reader. Jangan hanya mengandalkan opacity visual.

HTML Snippets — Siap Copy Paste

Semua class .btn membutuhkan base class + variant class. Ghost button menggunakan class berbeda (.btn-ghost-fixed) karena tidak memiliki border.

CSS States — Lengkap
CSS
/* Ghost — NO border */
.btn-ghost-fixed {
  display: inline-flex; align-items: center; justify-content: center;
  gap: 8px; padding: 10px 20px; border-radius: 8px;
  font-family: var(--font-primary); font-size: 14px; font-weight: 600;
  border: 1.5px solid transparent; /* transparent — no visible border */
  background-color: transparent; color: var(--color-text-secondary);
  transition: all 200ms ease-in-out; cursor: pointer;
  letter-spacing: .01em; white-space: nowrap; line-height: 1;
}
.btn-ghost-fixed:hover { background-color: var(--color-gray-100); color: var(--color-text-primary); }

/* Hover */
.btn-primary:hover {
  background-color: var(--color-primary-600); /* #1668B0 */
  border-color: var(--color-primary-600);
  box-shadow: 0 6px 20px rgba(30,127,214,.5);
  transform: translateY(-1px);
}

/* Pressed */
.btn-primary:active {
  background-color: var(--color-primary-700); /* #115188 */
  border-color: var(--color-primary-700);
  box-shadow: none;
  transform: translateY(1px) scale(.98);
}

/* Focus — keyboard nav only */
.btn:focus-visible, .btn-ghost-fixed:focus-visible {
  outline: none;
  box-shadow: 0 0 0 3px rgba(30,127,214,.35);
}

/* Selected */
.btn-primary[aria-pressed="true"],
.btn-primary.selected {
  background-color: var(--color-primary-800); /* #0D3C63 */
  border-color: var(--color-primary-800);
  box-shadow: inset 0 2px 4px rgba(0,0,0,.2);
}

/* Disabled */
.btn:disabled, .btn[disabled],
.btn-ghost-fixed:disabled {
  opacity: 0.4;
  cursor: not-allowed;
  pointer-events: none;
}
