Typography

Font families, sizes, weights, dan line heights untuk hierarchy dan readability

Typography Principles

Design Goals

  • Clarity: Teks harus mudah dibaca di semua ukuran
  • Hierarchy: Ukuran, weight, dan color membedakan konten penting
  • Consistency: Gunakan hanya typography scale yang telah ditentukan
  • Accessibility: Minimum font size 14px untuk body text, contrast ratio memenuhi WCAG AA

Font Weights

Light (300) - --font-weight-light

Untuk text yang kurang penting atau subtle

Normal (400) - --font-weight-normal

Default weight untuk body text

Medium (500) - --font-weight-medium

Untuk labels dan secondary headings

Semibold (600) - --font-weight-semibold

Untuk subheadings dan emphasis

Bold (700) - --font-weight-bold

Untuk headings dan strong emphasis

✅ Usage Guidelines

  • H1-H6: Gunakan font-weight-bold (700)
  • Labels: Gunakan font-weight-medium (500)
  • Body text: Gunakan font-weight-normal (400)
  • Secondary text: Gunakan font-weight-normal (400) + gray-500 color

Line Heights

Spacing vertikal antara baris teks untuk readability yang optimal.

Tight (1.2) - --line-height-tight
Gunakan untuk headings dan subheadings

Cocok untuk text yang singkat dan besar

Snug (1.375) - --line-height-snug
Good untuk smaller headings dan labels

Balanced spacing untuk readability

Normal (1.5) - --line-height-normal
Default untuk body text dan paragraf. Memberikan readability optimal.

Recommended untuk kebanyakan use case

Relaxed (1.625) - --line-height-relaxed
Untuk long-form content dan documentation.

Extra spacing untuk ease of reading

Loose (2) - --line-height-loose
Maximum spacing antara lines.

Gunakan hanya untuk special cases

Predefined Text Styles

Heading Styles

.text-h1 {
  font-size: var(--font-size-4xl); /* 36px */
  font-weight: var(--font-weight-bold);
  line-height: var(--line-height-tight);
  color: var(--color-text-primary);
}

<!-- H2 Styles -->
.text-h2 {
  font-size: var(--font-size-3xl); /* 30px */
  font-weight: var(--font-weight-bold);
  line-height: var(--line-height-tight);
}

Body Text Styles

<!-- Body Small -->
.text-body-sm {
  font-size: var(--font-size-sm); /* 14px */
  line-height: var(--line-height-normal);
}

Best Practices

✅ Do

  • Gunakan scale yang telah ditentukan, jangan buat custom sizes
  • Maintain font-size dan line-height yang sesuai untuk readability (minimum 14px untuk body)
  • Gunakan semantic HTML tags (h1-h6, p, strong, em)
  • Test readability pada dark dan light backgrounds
  • Ensure contrast ratio memenuhi WCAG AA (4.5:1 untuk normal text)

❌ Don't

  • Jangan buat custom font sizes - gunakan scale yang ada
  • Jangan gunakan font size kurang dari 12px untuk body text
  • Jangan mix font families dalam single component
  • Jangan gunakan text decoration lain dari underline untuk links
  • Jangan skip line-height untuk readability
