Icons

— icon SVG tersedia dalam 3 kategori — General, Sidebar Offline, dan Sidebar Online. Semua icon dipanggil via file path <img src>, tidak ada inline SVG.

Icon Guidelines

RULES

Aturan penggunaan icon untuk menjaga konsistensi, performa, dan kemudahan maintenance di seluruh produk GPOS Lite.

📌 Kapan Dipakai
  • Gunakan icon untuk memperkuat label — bukan menggantikannya
  • Cocok di button, input, navigasi, dan status indicator
  • Panggil via <img src> dengan path file yang benar
  • Gunakan ukuran sesuai konteks (small/medium/large/xlarge)
  • Sidebar icon: gunakan state Off dan On sesuai kondisi aktif
✅ Best Practice
  • Semua file icon berada di pages/assets/icons/
  • Jangan duplikasi icon ke folder lain
  • Selalu sertakan alt yang deskriptif
  • Tambahkan width dan height untuk mencegah layout shift
  • Ikuti naming convention yang sudah ada
✖ Hindari
  • Jangan embed SVG path langsung di HTML
  • Jangan gunakan icon tanpa alt text
  • Jangan ubah ukuran icon di luar token yang tersedia
  • Jangan letakkan icon baru di luar folder assets/icons/
  • Jangan gunakan icon sebagai pengganti teks di area penting

Icon Size Rules

4 ukuran TOKENS

Gunakan token ukuran yang konsisten. Jangan gunakan ukuran di luar daftar ini agar visual tetap harmonis di seluruh UI.

size=small
16px
--icon-size-small
size=medium
24px
--icon-size-medium
size=large
32px
--icon-size-large
size=xlarge
48px
--icon-size-xlarge
Token Px Tailwind Konteks Penggunaan
small 16px w-4 h-4 Badge, chip, helper text, inline icon
medium 24px w-6 h-6 Button, input, nav item (default)
large 32px w-8 h-8 Card header, empty state, feature icon
xlarge 48px w-12 h-12 Hero section, error page, onboarding

Implementation Code

CODE

Contoh implementasi icon dalam HTML standar, CSS Variable, dan Tailwind CSS.

HTML 
<!-- ============================================ -->
<!-- CARA MEMANGGIL ICON — DS GPOS Lite V2       -->
<!-- Path: pages/assets/icons/                   -->
<!-- ============================================ -->

<!-- Icon dasar — medium (24px, default) -->
<img src="../assets/icons/Icon:home.svg"
     alt="home"
     width="24"
     height="24">

<!-- Icon small (16px) -->
<img src="../assets/icons/Icon:Bell.svg"
     alt="notification"
     width="16"
     height="16">

<!-- Icon di dalam button -->
<button class="btn btn-primary">
  <img src="../assets/icons/Icon:plus.svg"
       alt="tambah"
       width="20"
       height="20">
  Tambah Item
</button>

<!-- Sidebar icon — state Off (tidak aktif) -->
<img src="../assets/icons/icon:sidebar offline:News:Off.svg"
     alt="berita"
     width="24"
     height="24">

<!-- Sidebar icon — state On (aktif) -->
<img src="../assets/icons/icon:sidebar offline:News:On.svg"
     alt="berita"
     width="24"
     height="24">
CSS 
/* ================================================
   ICON CSS VARIABLES — DS GPOS Lite V2
   Source: Design Tokens → Icons
   ================================================ */

:root {
  /* Icon Size Tokens */
  --icon-size-small:  16px;
  --icon-size-medium: 24px;  /* default */
  --icon-size-large:  32px;
  --icon-size-xlarge: 48px;
}

/* ── Utility Classes ────────────────────────── */

.icon {
  display: inline-block;
  width:  var(--icon-size-medium);
  height: var(--icon-size-medium);
  vertical-align: middle;
  flex-shrink: 0;
  object-fit: contain;
}
.icon-sm {
  width:  var(--icon-size-small);
  height: var(--icon-size-small);
}
.icon-lg {
  width:  var(--icon-size-large);
  height: var(--icon-size-large);
}
.icon-xl {
  width:  var(--icon-size-xlarge);
  height: var(--icon-size-xlarge);
}

/* ── Usage Example ──────────────────────────── */

.btn-icon {
  display: inline-flex;
  align-items: center;
  gap: 8px;
}
.btn-icon img {
  width:  var(--icon-size-medium);
  height: var(--icon-size-medium);
}
