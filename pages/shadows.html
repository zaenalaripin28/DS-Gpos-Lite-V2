Shadow System

6-level elevation system untuk depth, hierarchy, dan visual layering

Shadow Levels

Gunakan shadows untuk menunjukkan elevasi dan create visual hierarchy dalam design.

None (No Shadow)

shadow-none

Gunakan untuk flat design atau elemen tanpa depth

Small Shadow

shadow-sm / 0 1px 2px rgba(0,0,0,0.05)

Subtle shadow untuk light elevation

Medium Shadow (Default)

shadow-md

Default shadow untuk cards dan components

Large Shadow

shadow-lg

Untuk floating elements dan popovers

Extra Large Shadow

shadow-xl

Untuk modals dan dropdowns

2XL Shadow

shadow-2xl

Maximum depth untuk page overlays

Usage Guidelines

✅ Usage Chart

Shadow Use Case
shadow-sm Hover state, subtle lift
shadow-md Cards, buttons on hover
shadow-lg Floating panels, tooltips
shadow-xl Modals, dropdowns
shadow-2xl Full page overlays, dialogs

Code Examples

<!-- Card with default shadow -->
<div class="shadow-md p-4 rounded-lg">
  Card with medium shadow
</div>

<!-- Hover effect -->
<div class="shadow-sm hover:shadow-md transition-shadow">
  Elevate on hover
</div>

<!-- Modal -->
<div class="fixed inset-0 shadow-2xl">
  Full page modal
</div>
